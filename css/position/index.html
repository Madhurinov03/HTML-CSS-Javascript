Position:

  • Static
  • relative
  • fixed
  • absolute
  • sticky

static: It is static by default. It does not affected by left, top, right, bottom properties.

fixed::it stay in the same place even if the page is scrolled.Top,left,right,bottom properties are used to position the element.

relative: it adjust according to the left, top, bottom and right properties.

absolute: position absolute by default will move the position of an element based on screen size or body but if its ancestors have the position relative then it will adjust its position based inside ancestor.

sticky:it based upon user scroll position. It is positioned relative until the given offset position is met then it stick in place(like a fixed position).
